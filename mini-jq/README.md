# jq-mini

A minimal, educational reimplementation of a subset of [`jq`](https://stedolan.github.io/jq/) designed to be **small, readable, and easy to extend**.

`jq-mini` is **not** a drop-in replacement for `jq`. Its purpose is to demonstrate how a JSON query language works internally: parsing, ASTs, and evaluation.

### Primary Goals

- Provide a **minimal JSON query language**

### Non-Goals
